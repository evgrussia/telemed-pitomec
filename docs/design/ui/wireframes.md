# Wireframes: Телемед-Питомец

## Общая структура (Layout)

Приложение использует стандартный Telegram WebApp Layout:

- **Header**: Название раздела и кнопка закрытия/назад.
- **Main Content**: Скроллируемая область.
- **Bottom Navigation**: Таб-бар (Главная, ИИ, Врачи, Профиль).

---

## 2. Экран ИИ-диагностики (AI Diagnosis)

Цель: Удобная загрузка фото и получение быстрого результата.

```text
┌────────────────────────────────────────┐
│ [<]          ИИ-ДИАГНОСТИКА        [X] │
├────────────────────────────────────────┤
│  Выберите питомца:                     │
│  [ Барсик (кот) ▼ ]                    │
│                                        │
│  [ Сфотографируйте симптом ]           │
│  ┌──────────────────────────────────┐  │
│  │                                  │  │
│  │         [ 📷 Сделать фото ]      │  │
│  │         [ 🖼 Выбрать из галереи ] │  │
│  │                                  │  │
│  └──────────────────────────────────┘  │
│                                        │
│  Описание (опционально):               │
│  [ Вялость со вчерашнего вечера...   ] │
```
