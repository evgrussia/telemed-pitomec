# Телемед-Питомец — материалы для инвестора

**Однострочник:** Телеветеринария для владельцев домашних животных с ИИ-диагностикой в формате Telegram WebApp.

Документ даёт сжатый обзор проекта и ссылки на полную бизнес-документацию для due diligence.

---

## 1. Executive Summary

**Телемед-Питомец** — B2C-сервис в нише PetTech/HealthTech: Telegram WebApp, объединяющий **ИИ-диагностику по фото**, **видеоконсультации с ветеринарами 24/7**, **дневник здоровья питомца** и (на более поздних этапах) **маркетплейс** кормов и лекарств.

- **Проблема:** Владельцы животных не понимают серьёзность симптомов, не имеют доступа к ветеринару 24/7, теряют историю здоровья питомца. Рынок перегружен (дефицит врачей), очный приём дорогой и неудобный.
- **Решение:** Один продукт в Telegram: быстрая ИИ-оценка симптомов → при необходимости связь с лицензированным ветеринаром по видео → единая медкарта и напоминания.
- **Платформа:** 100% Telegram WebApp (без отдельного мобильного приложения) — низкий CAC, быстрый онбординг, встроенные платежи и push.
- **Модель:** Подписка (Basic/Premium) + комиссия с консультаций + перспектива B2B и маркетплейса.
- **Рынок:** 40M+ владельцев животных в РФ, TAM ветеринарных услуг ~50 млрд ₽/год, SAM телеветеринарии ~10–12 млрд ₽/год.
- **Стадия:** Discovery завершён; подготовка к Design и разработке MVP.

## 2. Рынок и возможность

| Показатель | Значение |
|------------|----------|
| **Домашние животные в РФ** | 74,8 млн (66% семей) |
| **Целевая аудитория** | 40M+ владельцев, фокус на городах-миллионниках |
| **TAM (ветуслуги)** | ~50 млрд ₽/год |
| **SAM (телемедицина для животных)** | ~10–12 млрд ₽/год |
| **Рост рынка ветуслуг** | +16,4% YoY; телеветеринария ~21% в год |
| **Рост PetTech/зоотоваров** | 510 млрд ₽ (2024), +17,7% YoY |

**Почему сейчас:** рост PetTech, зрелость Telegram (80M+ в РФ), доступность AI vision для диагностики, привычка к телемедицине после COVID, отсутствие доминирующего игрока с экосистемой «ИИ + врачи + история здоровья» в одном продукте.

Подробно: [Исследование рынка](research/market-research.md), [Конкурентный анализ](research/competitive-analysis.md).

## 3. Продукт

**Ключевые возможности (MVP и далее):**

- **ИИ-диагностика (AI-triage):** загрузка фото симптома → предварительная оценка и рекомендация (успокоить / наблюдать / к врачу) за минуты.
- **Видеоконсультации 24/7:** видеозвонок с лицензированным ветеринаром прямо в WebApp.
- **Дневник здоровья:** вес, питание, активность, история обращений и прививок.
- **Напоминания:** прививки, обработки, плановые осмотры через Telegram.
- **Маркетплейс (V1.2+):** корма, лекарства, доставка.

**Платформа:** React + Django, PostgreSQL, Computer Vision для анализа симптомов, WebRTC/сторонний провайдер для видео. Ветеринары — партнёры (не штат).

Подробно: [Видение продукта](product/vision.md), [PRD](product/prd.md), [User Stories](product/user-stories.md), [Дорожная карта](product/roadmap.md).

## 4. Бизнес-модель и финансы

**Источники дохода:**

1. **Транзакции:** комиссия 20–30% с платных консультаций (экспресс-чат, видеоконсультация, узкие специалисты).
2. **Подписка:** Basic (ограниченный ИИ + история), Premium (безлимит ИИ, скидки на консультации, приоритет) — 299–699 ₽/мес по текущим планам.
3. **Перспектива:** лиды в офлайн-клиники, маркетплейс (10–15% комиссии), B2B (страховые).

**Юнит-экономика (целевые показатели):**

| Метрика | Цель |
|---------|------|
| CAC | 350 ₽ |
| ARPU | ~1 200 ₽/год |
| LTV | ~1 800 ₽ |
| LTV/CAC | > 3x (цель 5.1x) |
| Payback | ~4 месяца |

**Точка безубыточности:** ~10 000 платных консультаций/мес или ~4 000 активных подписчиков Premium при OpEx ~2 млн ₽/мес. Прогноз достижения: 8–10 месяц после запуска MVP.

**Финансовые цели Year 1 (примерно):** к Q4 — 100K MAU, 15K платных транзакций/мес, выручка ~12 млн ₽/мес; целевой MRR к концу года — 5 млн ₽.

Подробно: [Финансовая модель](business/financial-model.md), [Стратегия монетизации](business/monetization-strategy.md), [Lean Canvas](business/lean-canvas.md).

## 5. Конкурентные преимущества

- **Единственная в РФ экосистема в Telegram:** ИИ-диагностика + живые консультации + дневник здоровья без установки отдельного приложения.
- **First-mover в нише AI-телеветеринарии в РФ.**
- **Дистрибуция через Telegram WebApp:** низкий CAC, виральность, встроенные платежи и уведомления.
- **Партнёрская модель с ветеринарами:** масштабируемый supply без содержания штата врачей.
- **Снижение тревоги и «лишних» визитов:** ИИ отсекает ложные тревоги и направляет к врачу при необходимости.

Подробно: [Конкурентный анализ](research/competitive-analysis.md), [Positioning & USP](marketing/positioning-usp.md).

## 7. Риски и правовая среда

**Ключевые риски:** точность ИИ (митигация — чёткие рекомендации и всегда возможность связи с врачом); регуляторика и лицензии (экспертиза до запуска, disclaimers, страхование); доступность ветеринаров-партнёров (ранние партнёрства, фокус на Москва/СПб); стабильность видеозвонков в WebApp (выбор провайдера, тесты).

Подробно: [Управление рисками](business/risk-management.md), [Юридическая совместимость](research/legal-compliance.md).
